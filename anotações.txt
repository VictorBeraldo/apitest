URI com recurso único: instancia
URI com + recursos: coleção
	Coleções e recursos:
		Limit:
			limite elementos
		Offset:
			pular offset elementos
		exemplo:	
			/postagens?anp=2018&limit=10&offset=30

CRUD: create read update delete
Status codes:
	200: succes
	500: server erro
	404: not found

header:
	content-type: tipo de metadados que espera o tipo do dado que espera do servidor
	ex:	
		text/xml
		application/json
